# Gambit Role Play Launcher

Приветствую. Этот репозиторий создан для публикации лаунчера, и автоматической подгрузки обновлений.. 

## Установка

Для установки лаунчера выполните следующие шаги:

1. Нажмите на последний релиз лаунчера.
2. Нажмите на gambit-launcher-setup-0.0.0.exe
3. Запустите установщик, выберите нужную папку и установите лаунчер.

## Сообщение об ошибках

Для того, чтоб сообщить об ошибке выполните следующие шаги:

1. Перейдите во вкладку [проблем](https://github.com/Gambit-RP/LauncherRelease/issues/new)
2. Опишите подробно ошибку, приложив логи с лаунчера и скриншот.
3. Нажмите submit new issue

## Просмотр логов

Для того, чтоб просмотреть логи выполните следующие шаги:

1. Зайдите в лаунчер, авторизируйтесь.
2. Нажмите вверху на значок настроек.
3. Нажмите кнопку открыть логи
