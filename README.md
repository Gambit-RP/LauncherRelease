# Gambit Role Play Launcher

Добро пожаловать! Этот репозиторий создан для публикации лаунчера и автоматической загрузки обновлений.

## Установка

Чтобы установить лаунчер, выполните следующие шаги:

1. Перейдите на страницу последнего релиза лаунчера.
2. Нажмите на файл `gambit-launcher-setup-0.0.0.exe`.
3. Запустите установщик, выберите папку для установки и завершите процесс установки лаунчера.

## Сообщение об ошибках

Если у вас возникла ошибка и вы хотите сообщить о ней, выполните следующие шаги:

1. Перейдите на вкладку [Issues](https://github.com/Gambit-RP/LauncherRelease/issues/new).
2. Опишите проблему подробно, прикрепите логи из лаунчера и скриншоты.
3. Нажмите кнопку "Submit new issue" для создания новой проблемы.

## Просмотр логов

Чтобы просмотреть логи, сделайте следующее:

1. Запустите лаунчер и войдите в него.
2. Нажмите на значок настроек в верхней части окна.
3. Затем нажмите кнопку "Открыть логи".
